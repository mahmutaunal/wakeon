Privacy Policy

Effective date: 2026-06-12

Wakeon is an open-source Wake-on-LAN application developed by AlpWare Studio.

Data Collection

Wakeon does not collect personal data.

Wakeon does not use analytics, advertising SDKs, tracking SDKs, or user accounts.

Local Data

Wakeon stores saved device information locally on your device, including:

  • Device name
  • MAC address
  • Broadcast address
  • Port
  • Device type
  • Favorite status
  • Last wake time

This data is not sent to AlpWare Studio or any external server.

Network Access

Wakeon uses local network access to:

  • Scan the local network
  • Send Wake-on-LAN magic packets
  • Help users configure devices

Network scan results stay on your device.

Backup and Restore

Wakeon allows users to export and import device backups as JSON files.

Backup files are created only when the user chooses to export them.

Users are responsible for storing and sharing their backup files securely.

Third-Party Services

Wakeon does not use third-party analytics or advertising services.

Open Source

Wakeon is open source. The source code is available on GitHub.
